Pemrograman Web

Materi Pemrograman Web

Pertemuan 1: HTML

HTML digunakan untuk membuat struktur halaman web.

<!DOCTYPE html>
<html>
<head>
    <title>Halaman HTML</title>
</head>
<body>
    <h1>Halo, ini halaman HTML!</h1>
</body>
</html>

Pertemuan 2: CSS

CSS digunakan untuk mempercantik tampilan web.

body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}

Pertemuan 3: JavaScript

JavaScript digunakan untuk membuat halaman web lebih interaktif.

<script>
alert("Selamat datang di website!");
</script>

Pertemuan 4: Pengenalan PHP

PHP digunakan untuk pemrograman server-side.

<?php
    echo "Hello, PHP!";
?>

Pertemuan 5: Mengontrol Alur Program

PHP memungkinkan penggunaan if-else untuk mengontrol alur program.

<?php
    $nilai = 75;
    if ($nilai >= 70) {
        echo "Lulus";
    } else {
        echo "Tidak Lulus";
    }
?>

Pertemuan 6: Perulangan dan Array

Perulangan digunakan untuk menjalankan kode berulang kali.

<?php
    $buah = array("Apel", "Mangga", "Jeruk");
    foreach ($buah as $b) {
        echo $b . "<br>";
    }
?>
